# Project Notes

This directory contains the project-specific text documents that were previously scattered at the repository root or under internal note folders.

Files:

- `algorithm.md`: current algorithm description aligned with the implementation.
- `constraint.txt`: project constraints and design boundaries.
- `experiments/mbpp_small_experiment_log.md`: running record of MBPP experiments and conclusions.

Operational configs and runnable YAML files now live under `trl/configs/` because they are part of the executable training/evaluation workflow.

Relevant config folders:

- `trl/configs/train/`
- `trl/configs/eval/`
- `trl/configs/comparison/`
